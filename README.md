# Python个人博客系统基于Django python209

## 项目概述

本项目是一款基于Python+Django的个人博客系统，设计新颖，技术先进，前端采用Html+Bootstrap5+jQuery技术，后端使用Django框架，数据库为Mysql8.2。

## 技术栈

- **总体架构**: Python + Django + Mysql
- **前端**: Html + Bootstrap5 + jQuery
- **后端**: Django
- **开发工具**: Pycharm，Navicat或Dbeaver

### 运行环境

- Python版本：3.11.4（建议3.9及以上）
- Django版本：4.2.4（建议Django3及以上）
- 数据库：Mysql8.2（8.0及以上）

## 功能模块

### 用户端

- **首页/博客展示**: 支持分页浏览，展示实时时间、最新文章、标签分类、搜索功能。
- **博文点赞**: 实时更新文章点赞数。
- **博文发布**: 填写标题、内容、分类、标签、封面图，发布新文章。
- **博文阅读/评论**: 阅读文章，查看评论，支持发表评论和回复，含敏感词检测。
- **留言板**: 查看留言时间分布图，可发布留言。
- **个人信息**: 查看和编辑个人资料。
- **修改密码**: 原密码验证后更新为新密码，含密码强度指示器。
- **登录注册**: 用户注册与登录，登录状态用于后续操作。

### 管理端

- 用户管理
- 文章管理
- 标签管理
- 评论管理
- 留言管理

## 系统角色

- 普通用户：浏览、评论、发布文章等。
- 管理员：管理用户、文章、标签、评论和留言。
